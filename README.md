# Comandos Git
| Comando                 | Descripcion                                                           | 
|-------------------------|-----------------------------------------------------------------------|
| git init                | Inicializa un repositorio GIT en un directorio local                  | 
| git clone <URL>         | Clona un repositorio GIT desde una URL remota                         | 
| git add [archivo]       | Agrega cambios de un archivo al área de preparacion o Stage           | 
|
